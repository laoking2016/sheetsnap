# Sheetsnap — 使命与愿景

## 一句话使命

**让每一份供应商报价单从附件到表格，只需一次点击。**

---

## 目标用户

中小型海外 B2B 企业的采购经理 / 采购助理（公司规模 10–200 人，年采购额 50 万–500 万美元）。

## 核心痛点

- 每周收到 10+ 份来自中国/东南亚供应商的 Excel 或 PDF 报价单
- ERP 系统老旧或根本无自动导入功能，必须手动逐行录入
- 报价单格式千奇百怪：产品编码/描述/单位/价格字段混乱
- 频繁录错，月底对账才发现，造成时间和订单损失

## 价值主张

> 采购经理时薪约 \$25–\$40，每张报价单手动录入需 5–15 分钟，相当于每次浪费 \$2–\$8。
>
> Sheetsnap 将时间压缩到 **几秒**，按次收费仅 **\$0.1–\$0.2**，价值感极强。

## 产品原则

1. **一次只做一件事，做到极致。** MVP 只做"报价单 → 标准 CSV"这一条链路，不分散精力。
2. **AI 是手段，不是目的。** 不自己训练模型，调用现成文档解析 API；价值在于 B2B 报价单场景的列映射和单位归一化。
3. **用户数据即信任。** 文件处理完即删，不出售数据，不做不必要的持久化。
4. **先跑通一个垂直品类（如电子元器件），再做深做透。** 不追求大而全。

## 不做（MVP 阶段明确放弃）

- ❌ 多行业模板学习（用户教系统规则）
- ❌ ERP / QuickBooks 等第三方系统自动同步
- ❌ PDF 扫描图片 OCR（手写/拍照的报价单）
- ❌ 多语言界面（只做英语）
- ❌ 团队协作 / 权限管理
- ❌ 企业定制 API 对接
